Biography

Building a website for teaching cooking is a fascinating project that also requires a harmonious blend of engaging content and a good user experience. The website needs to not only present cooking recipes clearly, accompanied by appealing food images and detailed instructional videos, but also be easy to navigate so users can quickly find suitable dishes based on ingredients, difficulty level, or preparation time. Designing a clean layout, using readable fonts, and employing colors that are close to the culinary world, along with a smart search system, will help users feel comfortable and interested in learning to cook. Additionally, fast navigation and interactive features are crucial for the website to become a trustworthy companion in everyone's kitchen.


Challenges

One of the biggest challenges when working with HTML and CSS is managing the layout and responsiveness across different devices. Even with Flexbox or Grid, sometimes two divs won't align as desired or automatically wrap to the next line at incorrect screen sizes. Particularly when trying to make the interface look good on both desktop and mobile/tablet devices, developers often have to test repeatedly with media queries, adjusting flex-wrap, min-width, max-width… Sometimes even a small whitespace or the box-sizing property can cause the layout to shift significantly, consuming a lot of time for debugging. Not to mention that CSS specificity can cause one class to override another, leading to styles not working as expected, forcing developers to use !important or rewrite entire rules.


Goals

The website aims to inspire people and share culinary knowledge in a simple and clear way, from beginners to those with experience, so everyone can be confident in their kitchen and enjoy the joy of cooking.
